# MUDWATER 师比画廊

让每一件原作，都成为可运营的 IP 资产

## 项目简介

MUDWATER 师比画廊是一个面向青年艺术家的原作交易与 IP 运营平台，致力于：
- 重建信任：严选入驻 + 第三方鉴证 + 明码标价
- 打开渠道：线上策展 + 数据推荐 + 内容引流
- 深度变现：版权授权居间 + 藏家变投资人 + 艺术家获持续收益

## 技术栈

- React 18
- Tailwind CSS 3
- Babel Standalone

## 快速开始

无需安装依赖，直接打开 `index.html` 即可运行。

```bash
# 克隆仓库
git clone https://github.com/gloryhonourmyheart/MUDWATER-ArtGallery.git

# 进入项目目录
cd MUDWATER-ArtGallery

# 使用浏览器打开
open index.html  # macOS
start index.html # Windows
```

## 项目结构

```
├── index.html    # 主页面（包含所有 React 组件）
```
